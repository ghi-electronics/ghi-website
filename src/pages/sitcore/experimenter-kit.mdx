## A structured course, not a quickstart

The Experimenter Kit is designed for **learning the platform** — not for evaluating it for a commercial project. Each experiment builds on the one before, introducing concepts in an order designed for retention. The companion eBook reads like a textbook, not a README.

**What you'll do as you work through the course:**

- Read real sensors
- Control LEDs and motors
- Drive displays
- Talk to peripherals over GPIO, ADC, PWM, and signal decoders
- Build connected devices that publish data

## What's in the kit

<div className="split">

![BrainBot robot with FEZ Bit](/img/experimenter-brainbot.png)

<div className="split__text">

### FEZ Bit, BrainClip, and BrainBot

The SITCore **FEZ Bit** is the brain of every project — deploy and debug from Microsoft Visual Studio using the included USB cable.

The **BrainClip** accessory pack adds multiple circuit modules that connect to the FEZ Bit using alligator clip wires. GPIO, ADC, PWM, signal decoders — the variety lets you cover many embedded concepts without rewiring.

With minimal assembly, the **BrainBot** robot adds onboard sensors and motor control — the moment code starts moving real hardware is the moment the concepts click.
