## No experience necessary

The Experimenter Kit lets engineers explore SITCore and TinyCLR OS without designing hardware. Within minutes, you'll run code that interacts with real sensors, displays, and peripherals — each experiment building on the previous one, gradually introducing more advanced concepts.

**Experiments included:**

- Reading sensors
- Controlling LEDs and motors
- Interacting with displays
- Communicating with peripherals
- Building connected devices

## What's in the kit

<div className="split">

![BrainBot robot with FEZ Bit](/img/experimenter-brainbot.png)

<div className="split__text">

### FEZ Bit, BrainClip, and BrainBot

The SITCore **FEZ Bit** is the brain of every project — deploy and debug from Microsoft Visual Studio using the included USB cable.

The **BrainClip** accessory pack adds multiple circuit modules that connect to the FEZ Bit using alligator clip wires. GPIO, ADC, PWM, signal decoders — the variety lets you experiment with many embedded concepts without re-wiring.

With minimal assembly, the **BrainBot** robot adds onboard sensors and motor control — the moment your code starts moving real hardware.
