## When to choose Development Boards

<div className="decision-callout">
  <div className="decision-callout__yes">
    <h3>Pick a Dev Board when</h3>
    <ul>
      <li>You're evaluating SITCore for a commercial project</li>
      <li>You need every peripheral exposed and accessible</li>
      <li>You want debug headers, click connectors, and a known platform</li>
      <li>You're starting production-realistic firmware on something stable</li>
    </ul>
  </div>
  <div className="decision-callout__no">
    <h3>Consider something else if</h3>
    <ul>
      <li>You're learning the platform — <a href="/sitcore/experimenter-kit/">Experimenter Kit</a></li>
      <li>You're prototyping in a maker form factor — <a href="/sitcore/sbc/">Single Board Computer</a></li>
      <li>You're going to production — <a href="/sitcore/som/">System on Module</a></li>
      <li>You're designing your own board — <a href="/sitcore/soc/">System on Chip</a></li>
    </ul>
  </div>
</div>
