## Powered by TinyCLR OS

<div className="centered-image">

![TinyCLR OS](/img/tinyclr-logo.png)

</div>

The SITCore family runs **TinyCLR OS** — a royalty-free embedded operating system with an integrated Common Language Runtime that executes compiled C# code on every SITCore module. Full Visual Studio integration, generics, real debugging, and a security stack built in.
