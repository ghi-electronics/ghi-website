## Years of Innovation

<div className="centered-image">

![GHI Electronics history](/img/history-logo.png)

</div>

From the first FAT file system chipset on the market in 2003 to the DUELink ecosystem of today, here's a look at the milestones that shaped GHI Electronics.

<div className="timeline">
  <div className="timeline-item">
    <div className="timeline-year">2026</div>
    <div className="timeline-body">Empower educators by expanding the DUELink ecosystem with MicroBlocks, a block-coding option.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2025</div>
    <div className="timeline-body">Introduced DUELink, an electronic-module building-blocks ecosystem that works with the world's most popular microcomputers and coding languages.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2024</div>
    <div className="timeline-body">Endpoint extended with full Python 3 support through VS Code.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2023</div>
    <div className="timeline-body">Introduced Endpoint, the world's first full-scale .NET system-on-modules.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2022</div>
    <div className="timeline-body">Introduced DUE, a scripting language that is easy to learn.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2021</div>
    <div className="timeline-body">Expanded SITCore with the world's smallest, lowest-power, commercial-grade .NET C# SC13 chipsets and boards.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2020</div>
    <div className="timeline-body">Introduced SITCore (Secure IoT Core) SC20 chipsets and modules. Refreshed the FEZ product line with all-new SITCore SBCs.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2019</div>
    <div className="timeline-body">Introduced TinyCLR 2.0, a world-class secure .NET IoT operating system.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2018</div>
    <div className="timeline-body">Released the ultra low-cost FEZ for the maker and educational markets. Introduced the Universal Compute Module standard. Released TinyCLR OS, bringing .NET to embedded devices.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2017</div>
    <div className="timeline-body">Invented a universal compute module standard. Introduced TinyCLR OS, a modern, portable, and extendable .NET embedded operating system.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2016</div>
    <div className="timeline-body">Entered into a partnership with the BeagleBoard.org Foundation to become an official manufacturer. Became a design and manufacturing launch partner for the BeagleBone Black Wireless.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2015</div>
    <div className="timeline-body">Introduced the G30 and G80 SoCs (System-on-Chip), the world's smallest and lowest-cost .NET Micro Framework chipsets on the market.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2014</div>
    <div className="timeline-body">Introduced the ALFAT-2 FAT file system chipset update with SD-Reader mode.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2013</div>
    <div className="timeline-body">Introduced G400-D and G400-S SoMs, the most advanced NETMF products on the market.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2012</div>
    <div className="timeline-body">ALFAT FAT file system chipset updated with modern processor, built-in USB Host, and Microsoft Long Filename license. Introduced FEZ Hydra and G120 SoM.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2011</div>
    <div className="timeline-body">Partnered with Microsoft on .NET Gadgeteer technology. Introduced FEZ Spider, the first .NET Gadgeteer mainboard.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2010</div>
    <div className="timeline-body">Introduced the .NET Micro Framework-based FEZ (Freakin' Easy) product line, targeted at makers and educators.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2009</div>
    <div className="timeline-body">Introduced EMX and ChipworkX SoMs running .NET Micro Framework with advanced features like SQLite.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2008</div>
    <div className="timeline-body">Signed a partnership with Microsoft for developing .NET Micro Framework devices. Introduced the Embedded Master SoM.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2007</div>
    <div className="timeline-body">Introduced uALFAT, the first FAT file system chipset with Microsoft-patented Long File Name (LFN) technology.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2006</div>
    <div className="timeline-body">Introduced ALFATxp, the first FAT file system chipset with Java VM.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2005</div>
    <div className="timeline-body">Introduced USBwiz, the first FAT file system chipset with USB Mass Storage support.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2004</div>
    <div className="timeline-body">ALFAT upgraded to be the first FAT chipset with over 50 KB per second file access speed.</div>
  </div>
  <div className="timeline-item">
    <div className="timeline-year">2003</div>
    <div className="timeline-body">Introduced ALFAT, the first FAT file system chipset on the market.</div>
  </div>
</div>
