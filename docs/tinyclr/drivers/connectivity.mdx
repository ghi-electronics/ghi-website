title: Connectivity Drivers
---

Connectivity drivers extend TinyCLR's [networking core](../feature/networking-core.mdx) with hardware-specific utilities for radios and modules — currently the Microchip WINC1500/1510 WiFi modules.

See also: the network and W5500 drivers documented under [Software Utilities](software-utility.mdx#basic-network).

## WINC15x0

![WINC15x0](/img/docs/drivers/WINC15x0.png)

Utility package that exposes the additional services the **Microchip ATWINC1500/1510** WiFi module provides on top of the base WiFi support — access point scanning, RSSI, MAC address, firmware version, and over-the-air firmware updates.

See the [WiFi](../feature/wifi.mdx) page for the WiFi setup and the [WINC1500 utilities section](../feature/wifi.mdx#winc1500-utilities) for the full API.

NuGet package: **`GHIElectronics.TinyCLR.Drivers.Microchip.Winc15x0`**.
