Drivers for external memory chips — EEPROMs, flash, and similar.

## AT25M02

![AT25M02](/img/docs/drivers/AT25M02.png)

The **AT25M02** is a 2 Mbit SPI **EEPROM** from Microchip. The driver exposes simple `Read` and `Write` methods for byte-level access.

NuGet package: **`GHIElectronics.TinyCLR.Drivers.Microchip.AT25M02`**.
