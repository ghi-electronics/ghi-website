Drivers for motor control — servos, steppers, and DC motors.

## Servos

![Servo](/img/docs/drivers/servo.png)

Both **positional** servos (fixed-angle positioning) and **rotational** (continuous-rotation) servos are supported. See the [PWM](../feature/pwm.mdx#servo-motors) page for the full wiring and code example.

NuGet package: **`GHIElectronics.TinyCLR.Drivers.Servo`**.
