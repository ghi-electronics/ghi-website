TinyCLR runs on **SITCore** — a family of secure embedded hardware from GHI Electronics, designed and manufactured in the USA. The runtime, the development tools, and the C# code you write stay the same across every form factor. Pick the level of integration that matches where your project is.

## Pick a form factor

As you move from chip to complete board, you trade unit cost and customization for time-to-first-blink and on-board peripherals. All four run the same TinyCLR firmware.

| If you're… | Use a… |
|---|---|
| Designing a custom PCB and want the lowest unit cost | [System on Chip](soc.mdx) |
| Designing a carrier board but want CPU, memory, and networking pre-validated | [System on Module](som.mdx) |
| Prototyping or shipping in low volume, want a maker form factor | [Single Board Computer](sbc.mdx) |
| Evaluating TinyCLR with display, peripherals, and debug headers in the box | [Development Board](dev.mdx) |
| Learning TinyCLR with a structured curriculum, sensors, and a robot | [Experimenter Kit](experimenter-kit.mdx) |

## System on Chip

[![SITCore SC20100S](/img/docs/system-on-chip.png)](soc.mdx)

TinyCLR on a bare chipset. Drop a SITCore SoC onto your own PCB and you get the full .NET feature set at the smallest footprint and lowest unit cost. Available as 48-pin QFN, 100-pin LQFP, and 265-ball BGA packages.

[**Go to System on Chip →**](soc.mdx)

## System on Module

[![SITCore SoMs](/img/docs/module-options-size.png)](som.mdx)

Pre-validated computing modules — surface-mount or 200-pin SO-DIMM. CPU, memory, networking, and the TinyCLR runtime in one package, ready to drop onto a custom carrier board.

[**Go to System on Module →**](som.mdx)

## Single Board Computer

[![SITCore SBC line](/img/docs/sitcore-sbc-line.png)](sbc.mdx)

Maker-friendly boards in standard form factors — Arduino Uno, Raspberry Pi Pico, Adafruit Feather, BBC micro:bit, Seeeduino Xiao. Plug in USB, open Visual Studio, deploy. No JTAG, no external programmer, no extra toolchain.

[**Go to Single Board Computer →**](sbc.mdx)

[![SITCore Dev Boards](/img/docs/sitcore-dev-boards.png)](dev.mdx)

Complete evaluation platforms with display, peripherals, mikroBUS click connectors, and debug headers. The serious starting point for teams evaluating TinyCLR for a commercial product.

## Experimenter Kit

[![SITCore Experimenter Kit](/img/experimenter-kit-hero.png)](experimenter-kit.mdx)
