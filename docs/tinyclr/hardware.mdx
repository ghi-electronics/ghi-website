TinyCLR runs on a family of secure embedded hardware called **SITCore** — chipsets, system-on-modules, single-board computers, and dev boards from GHI Electronics, designed and manufactured in the USA. The runtime, the developer experience, and the C# code you write all stay the same across the family. Pick the form factor that fits your product.

## System on Chip

[![SITCore SC20100S](/img/docs/system-on-chip.png)](soc.mdx)

Drop a TinyCLR-capable chipset onto your own PCB. The smallest, lowest-cost path to a custom product. [**Learn more →**](soc.mdx)

## System on Module
