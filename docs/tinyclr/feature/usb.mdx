Being the go-to for modern communication, TinyCLR includes a wide range of USB (Universal Serial Bus) services, for both [USB Client](usb-client.mdx) and [USB Host](usb-host.mdx). The USB Client is the default Visual Studio interface, allowing for easy deploying and troubleshooting.
