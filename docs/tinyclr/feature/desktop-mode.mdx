**The emulator** runs a TinyCLR application on a Windows PC against the real CLR core — the *same* C# that deploys to the device, with no hardware attached. It's for developing, debugging, and testing application logic on the PC, where breakpoints and fast edit-run cycles are available.

Press **F5** in Visual Studio to run and debug on the PC, then deploy the unchanged project to a SITCore device when you're ready. The emulator replaces the previous Desktop (.NET Framework 4.8) mode.

:::note
The emulator is new and still being validated — not every device feature is covered yet. Use real hardware to confirm anything that depends on the device or its peripherals.
:::
