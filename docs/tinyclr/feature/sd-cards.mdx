---
title: SD/MMC Cards
---

SD and MMC cards are fully supported through the [File System](file-system.mdx) libraries.

This is a sample on how setup the SD driver.

```csharp
var sd = StorageController.FromName(SC20100.StorageController.SdCard);

var drive = FileSystem.Mount(sd.Hdc);
```

Additionally, SPI SD drivers are also supported through the `ManagedFileSystem` software utility [drivers](https://docs.ghielectronics.com/software/tinyclr/drivers/software-utility.html).
