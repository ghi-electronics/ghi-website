<h1 className="api-package-heading">System.Device.I2c</h1>

**NuGet:** `GHIElectronics.TinyCLR.Devices.I2c`

:::info
The standard, .NET-compatible `System.Device.I2c` API for TinyCLR. It ships inside the **[GHIElectronics.TinyCLR.Devices.I2c](../GHIElectronics.TinyCLR.Devices.I2c/index.mdx)** NuGet — there is no separate package to install.
:::
