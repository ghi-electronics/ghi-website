## WakeupPin

Hardware wake-up pin bit flags for `GHIElectronics.TinyCLR.Native.Power.Shutdown(int, System.DateTime)`. OR-combine to allow any of the selected pins to wake the device. Bit position matches the STM32H7 PWR_WAKEUP_PINx slot index (bit N → WKUP(N+1)).

| Member | Value |
|---|---|
| `PA0` | `1 << 0` |
| `PA2` | `1 << 1` |
| `PI8` | `1 << 2` |
| `PC13` | `1 << 3` |
| `PI11` | `1 << 4` |
| `PC1` | `1 << 5` |
