<h1 className="api-package-heading">System.Device.Gpio</h1>

**NuGet:** `GHIElectronics.TinyCLR.Devices.Gpio`

:::info
The standard, .NET-compatible `System.Device.Gpio` API for TinyCLR. It ships inside the **[GHIElectronics.TinyCLR.Devices.Gpio](../GHIElectronics.TinyCLR.Devices.Gpio/index.mdx)** NuGet — there is no separate package to install.
:::
