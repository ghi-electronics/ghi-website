![Download](/img/docs/downloads.png)

Most users only need the **firmware** download below — your IDE applies it to the device after the rest of the setup is in place. Downloads are either **Prerelease** (newest builds; APIs may still change, issues possible) or **RTW** (*Ready to Wear* — stable and ready to ship in commercial products).

:::tip
**You probably don't need most of this page.** Most users need only the firmware and TinyCLR Config (below). The Visual Studio extension installs from the Visual Studio Marketplace, libraries are pulled via NuGet, and most projects never download anything else. The remaining downloads are provided as a backup — for offline installs, locked-down corporate environments, or specific older versions.
:::

## Firmware

TinyCLR is the firmware that lives on your hardware. The firmware version loaded on the hardware must match the version of the Visual Studio extension and the libraries you build against.
