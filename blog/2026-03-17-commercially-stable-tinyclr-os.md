TinyCLR has been running .NET on thousands of products in industries around the world. With a clean year since the previous release and minimal issues filed, we are publishing a commercially-stable refresh — version 2.4.0.1000.

{/* truncate */}

## What's new in 2.4.0.1000

**HDMI at higher resolution** — A 120 MHz SDRAM overclock option is now available, enabling 1280×800 output for customers who need it.

**WPA/WPA2 on both AP and Client** — Wi-Fi mode now supports WPA/WPA2 security in both access-point and client configurations.

**TLS and HttpListener fixes** — Secure network connection improvements address a connection issue with HttpListener.

Visual Studio 2026 is fully supported.
