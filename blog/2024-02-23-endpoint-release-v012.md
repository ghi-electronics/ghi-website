Endpoint v0.1.2 ships four new capabilities: OpenCV support, custom boot screen, faster deployment, and Avalonia UI support.

{/* truncate */}

**OpenCV** integration is experimental in this release — GHI Electronics is actively collecting use case feedback for AI on a .NET-focused device. If you have an application in mind, share it on the [forum thread](https://forums.ghielectronics.com/t/endpoint-release-v0-1-2/24891).

**Avalonia UI** brings a cross-platform .NET UI framework to Endpoint, enabling rich graphical applications on the device display.

**Custom boot screen** and **deployment speed improvements** round out the release.
