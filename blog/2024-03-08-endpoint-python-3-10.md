![Endpoint Python 3.10](/img/press/endpoint-python.jpg)

Endpoint v0.1.3 adds Python support — not MicroPython, but the full CPython 3.10 implementation. Every standard library, every package from PyPI, running directly on Endpoint hardware.

{/* truncate */}

The Endpoint platform already supports full .NET 8. Python 3.10 extends that reach to the large community of developers who work in Python, giving both ecosystems access to the same hardware, the same peripherals, and the same remote deploy and debug workflow from VS Code.
