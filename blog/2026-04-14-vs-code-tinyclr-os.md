GHI Electronics has released a VS Code extension for TinyCLR OS, bringing IntelliSense, deploy, and step-through debug to every SITCore board — from the compact FEZ Stamp to full System on Modules.

{/* truncate */}

The extension works the same way developers expect from the Visual Studio counterpart: write C#, hit deploy, set breakpoints, and inspect variables on real hardware. The extension currently runs on Windows.

## Community spotlight: Tiny MQTT Broker

Developer Skeller used the release as an opportunity to port the Eclipse Paho M2Mqtt broker library to TinyCLR OS, making a fully managed MQTT broker available for SITCore devices. The project is open source and available on [GitHub](https://github.com/skeller2015/TinyMQTTBroker_v1).
