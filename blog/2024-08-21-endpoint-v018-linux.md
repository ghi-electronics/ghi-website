Endpoint v0.1.8 extends the platform beyond the Endpoint Domino board to other Linux-based hardware. The release has been tested on BeagleBone Black and Raspberry Pi 4.

{/* truncate */}

Use Endpoint Config to automatically install .NET on a compatible board, then deploy and debug .NET applications using the Endpoint extension in Visual Studio or VS Code — the same workflow as the Endpoint Domino.

For production use, the official Endpoint Domino board offers broader peripheral support — CAN, ADC, digital signal, RTC, USB, SD, display, IFU, watchdog, and camera — and boots in under 5 seconds.
